# Personal Portfolio 🚀

Mi portafolio personal de desarrollo web, diseñado para presentar mis proyectos, habilidades técnicas y trayectoria de forma limpia y eficiente.

## 📌 Características

- **Diseño Adaptativo:** Interfaz totalmente responsiva y optimizada para todo tipo de pantallas mediante CSS moderno.
- **Renderizado Dinámico:** Sección de proyectos inyectada dinámicamente mediante JavaScript (Vanilla JS), facilitando el mantenimiento y la escalabilidad del sitio.
- **Estética Limpia:** Enfoque visual moderno con efectos de desfoque (Glassmorphism) y paleta de colores optimizada para una lectura cómoda.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica avanzada.
- **CSS3:** Estilos personalizados, Flexbox, CSS Grid y variables globales para una arquitectura limpia.
- **JavaScript (ES6+):** Manipulación dinámica del DOM para la carga de proyectos.
